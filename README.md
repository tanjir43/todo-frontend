# todo-frontend

## Clone Project

```sh
git clone https://github.com/tanjir43/todo-frontend.git
cd todo-frontend
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests

```sh
npm run test:unit
```
